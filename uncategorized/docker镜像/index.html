#docker 操作

mysql

获取镜像

运行容器

$PWD 为当前目录，可当变量使用

  • -d：在后台运行
  • -p：容器的 3306 端口映射到宿主机 13306
  • -v：容器的 /etc/mysql/conf.d 配置文件映射到宿主机 $PWD/mysql/conf
  • -e：mysql root 用户的密码

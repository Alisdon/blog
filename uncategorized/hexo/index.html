hexo

2018-07-28 Comments

安装hexo

npm install -g hexo-cli

初始化项目

hexo init <folder>
cd <folder>
npm install``

启动项目

hexo server

更换主题

选择喜欢的主题并根据选择的主题进行更换，并将_config.yml中的theme主题替换成在文件夹theme中的文件名称, 添加主题中需要的图片等素材

hexo快捷命令及快速发布

hexo clean 清理静态化内容
hexo generate 简化 hexo g
hexo deploy 简化 hexo d

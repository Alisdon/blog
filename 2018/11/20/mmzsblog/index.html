用于mmzsblog网站博客开发

十一月 20, 2018

安装nodejs

前往 node官方下载地址下载对应版本进行安装

安装git

前往 git官方下载地址下载对应版本进行安装

安装hexo

$ npm install -g hexo-cli

此步骤需要将nodejs安装成功后才可进行，否则将报npm命令不存在的错误

git克隆仓库模版代码

$ git clone https://www.mmzsblog.cn/mmzs/blog.git

此步骤需要将git安装成功后才可进行，否则将报git命令不存在的错误

安装依赖文件

$ cd blog
$ npm install

执行成功上述操作将出现图中文件

新建文章

创建文章之前先更新仓库代码，然后再创建一篇新文章

$ git pull
$ hexo new 你的文章名称

打开当前文件夹下的source/_post目录，找到新建的文件（一般是根据上方新建的文章名称作为文件名称）进行编写

静态图片的处理

  1. 放到source/img文件中，在文章中需要进行![](/img/xxx.jpg)的引入
  2. 直接放到和文章名称相同的文件夹中，在文章中需要进行![](./xxx.png)的引入

发布文章

由于mmzs是通过github进行重定向，所以将不做清理静态文件的操作，避免提交后需要重新设置的重复操作

打包编译

$ hexo generate      简写        hexo g

发布提交

$ hexo deploy         简写        hexo d

如果出现以上提示则说明提交成功，隔几分钟后刷新https://blog.mmzsblog.cn即可查看刚刚新增的文章
